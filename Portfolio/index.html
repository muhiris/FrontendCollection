SwiftDelivers - Food Delivery Mobile Application

Project Link

A delivery based mobile application that offers food for delivery and pickup from different restaurants across some of the big cities of Pakistan.

Features
  • Users can signup and use the credentials to login to the app.
  • Location input for the users to input their desired location. It then displays the restaurants situated in that area.
  • Users can also filter restaurants based on the services they provide i.e. delivery or pickup.
  • Users can add multiple items to the cart and get to know the subtotal price.
  • A checkout screen for the user to review the selected items and confirm the order. Users also need to provide their address and specify the payment method.
  • Order confirmation screen to display the confirmation animation and the approximate time for delivery.
